## 📑 연구 개요
- Spring WebFlux와 Spring MVC의 아키텍처 차이 분석
- MySQL(RDB)과 Redis(NoSQL)를 사용하여 다양한 동시 사용자 수(100~5000명)에 따른 응답 시간을 측정
- 실험 결과, WebFlux 기반 시스템이 높은 동시성을 처리하는 데 유리, NoSQL이 대량 데이터 처리에서 우수한 성능 보임
